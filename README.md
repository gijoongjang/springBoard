# springboot project
springboot 이용한 게시판 만들기

<br>

## 사용기술
Java, Spring Boot, Spring Security, MyBatis, JSP, JavaScript, jQuery, MySQL

<br>

## 구현기능
### 공통
- 회원가입
- 로그인(User/Admin)
### User
- 게시판 등록, 조회, 수정, 삭제
- 게시판 파일 다중 업로드
- 쿠키 이용한 조회수 체크
### Admin
- 회원 삭제
